## üí° KEY INSIGHTS FROM REPORTS

### Common Missing Items Across All Reports

1. **Moisture Testing** (Reports 1, 4, 6)
   - Required by IICRC S500
   - Often omitted in estimates
   - Critical for mold prevention
   - Cost: $150-400

2. **Antimicrobial Treatment** (Reports 1, 4, 6)
   - Required for Category 2/3 water
   - Standard practice after contamination
   - Prevents bacterial/mold growth
   - Cost: $100-350

3. **Roof Deck Inspection** (Reports 2, 5)
   - Critical for storm damage
   - Often "hidden" until tear-off
   - Can add 10-20% to roof cost
   - Cost: $3,500-15,000

4. **Electrical Inspection** (Reports 1, 3, 5)
   - Required after water/fire damage
   - Safety-critical item
   - Often overlooked
   - Cost: $150-500

5. **Structural Engineering** (Reports 2, 5)
   - Required for commercial properties
   - Required in many jurisdictions for storm damage
   - Provides liability protection
   - Cost: $1,000-3,500

6. **Permit Fees** (Reports 2, 3, 5)
   - Required for most major repairs
   - Often omitted from estimates
   - Varies by jurisdiction
   - Cost: $500-4,000

### Severity Level Definitions

**Error (Red Flag)**
- Item is required by code or industry standard
- Omission creates safety risk or code violation
- Likely to cause claim dispute or supplement
- Must be addressed before project completion

**Warning (Yellow Flag)**
- Item is standard practice but not strictly required
- Omission may cause quality or longevity issues
- Should be discussed with adjuster
- Recommended for complete restoration

**Info (Blue Flag)**
- Item is best practice or value-add
- Omission won't prevent functional repair
- May improve outcome or reduce future risk
- Consider based on budget and priorities

## üîß TECHNICAL IMPLEMENTATION

### Database Schema
Reports are stored in the `reports` table with JSONB structure:

```sql
CREATE TABLE reports (
  id UUID PRIMARY KEY,
  user_id UUID REFERENCES users(id),
  team_id UUID REFERENCES teams(id),
  estimate_name TEXT NOT NULL,
  estimate_type TEXT,
  damage_type TEXT,
  result_json JSONB NOT NULL,  -- Full analysis structure
  paid_single_use BOOLEAN DEFAULT FALSE,
  created_at TIMESTAMPTZ DEFAULT NOW(),
  expires_at TIMESTAMPTZ
);
```

### Querying Reports

**Get all reports for a team:**
```sql
SELECT * FROM reports WHERE team_id = 'your-team-id' ORDER BY created_at DESC;
```

**Get high-risk reports:**
```sql
SELECT * FROM reports WHERE result_json->>'risk_level' = 'high';
```

**Get reports with missing items:**
```sql
SELECT 
  estimate_name,
  jsonb_array_length(result_json->'missing_items') as missing_count
FROM reports
WHERE jsonb_array_length(result_json->'missing_items') > 0
ORDER BY missing_count DESC;
```

**Get total missing value:**
```sql
SELECT 
  estimate_name,
  (result_json->'total_missing_value_estimate'->>'low')::numeric as missing_low,
  (result_json->'total_missing_value_estimate'->>'high')::numeric as missing_high
FROM reports
ORDER BY missing_high DESC;
```

**Get reports by damage type:**
```sql
SELECT * FROM reports WHERE damage_type = 'water_damage';
```

### API Integration

Reports can be accessed via your API:

